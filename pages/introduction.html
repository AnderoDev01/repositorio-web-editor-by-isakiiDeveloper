Introdução ao CCode

Bem-vindo ao mundo do CCode! Esta game engine foi projetada para tornar o desenvolvimento de jogos acessível e divertido, utilizando um sistema de programação visual baseado em blocos.

Com o CCode, você pode:

  • Criar jogos 2D de diversos gêneros.
  • Desenvolver aplicativos interativos.
  • Aprender os fundamentos da lógica de programação de uma maneira intuitiva.
  • Exportar seus projetos para diferentes plataformas (ex: APK para Android).

Esta Wiki serve como um manual completo para você explorar todas as funcionalidades do CCode, desde os blocos mais básicos até os conceitos mais avançados.

Principais Características

  • Programação Visual por Blocos: Arraste e solte blocos para construir a lógica do seu jogo, sem precisar escrever código complexo.
  • Modos de Dificuldade: Um Modo Simplificado (Noob) para iniciantes e um Modo Normal com acesso total às funcionalidades.
  • Editor de Níveis Integrado: Crie e gerencie os níveis dos seus jogos diretamente na engine.
  • Motor de Física: Adicione comportamentos realistas aos seus objetos com gravidade, colisões e mais.
  • Suporte a Recursos: Importe facilmente sprites, sons, fontes e outros assets para seus projetos.

Pronto para começar? Navegue pelas seções desta Wiki para aprender mais!
